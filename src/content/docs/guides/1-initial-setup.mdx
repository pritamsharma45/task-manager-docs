When you first open the application, you will find that it is pre-populated with dummy data.
You can try it out with the existing data, or you can delete all the data and begin using it as a blank application.

The first step is to save your application preferences.
Go to the Settings page where you will find three tabs - Preferences, Role Setting, and Advanced.

You can also import your previous clients into this application.
Read [this guide](/guides/) to see how it's done.

## Save Application Preferences

- When you start using this application, you should first save your application preferences such as your company's profile, tax settings, etc.

![Application Preferences](../../../assets/guides/initial-setup/application-prefs.png)

## Try it out with dummy data

- Navigate to Settings > Advanced.
- Click on the switch to enable below buttons.
- Click on Test with dummy data.
- Proceed with the confirmation prompt.
  ![test-with dummy data](../../../assets/guides/initial-setup/test-with-dymmy-data.png)

## Delete App Data

- Navigate to Setting > Advanced
- Click on the switch that says Enable below buttons.
- You will be presented with two buttons - `Delete all` & `Delete all except user data`.

### Delete app data except for user data and app preferences

- Navigate to Setting > Advanced.
- Click on the switch to enable the buttons below.
- Click on Delete and then choose Delete All data except the user's data.
- Proceed with the confirmation prompt

### Delete entire application data

- Navigate to Settings> Advanced
- Click on Delete and then choose Delete All
- Proceed with the confirmation prompt
  ![Delete app screenshot](../../../assets/guides/initial-setup/delete-app.png)

## Import your clients

- Navigate to Settings> Advanced
- Click on Pick CSV or Excel file.
  :::note
  You should first download a sample CSV file to know the appropriate column fields. You can download it and then add your clients to it, keeping the column field header names the same, and then upload it.
  :::
  ![Import clients](../../../assets/guides/initial-setup/import-clients.png)
